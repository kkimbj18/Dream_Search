### 1. 개요
프론트엔드와 백엔드를 혼자 개발한 1인 토이프로젝트.<br />
데이터 처리, 에러 처리, 서버 배포 등을 <br />
직접 개발해보면서 확실히 다지기 위해 진행한 프로젝트다.

### 2. 개발 환경 및 기술

**Front-End**
- React.js

**Back-End**
- Node.js + Express.js
- MongoDB + Mongoose
- Naver Cloud Platform
