### 1. 개요
프론트엔드와 백엔드를 혼자 개발해보면서 상호간에 어떻게 데이터를 처리하고 넘겨야 좋을지부터, 에러 발생 확인 및 처리를 위해 어떤 작업을 각 파트에서 해줘야하는지 등을 
직접 깨닫기 위해 진행한 토이 프로젝트다.

### 2. 개발 환경 및 기술
**IDE**
- VS Code
**Front-End**
- React.js
**Back-End**
- Node.js + Express.js
- MongoDB + Mongoose
- Naver Cloud Platform
