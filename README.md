### 1. 개요
1인 토이프로젝트<br />
데이터 처리, 에러 처리, 서버 배포 등을 직접 개발해보면서 확실히 다지기 위해 진행한 프로젝트입니다. <br />
서로 같은 목표(BE 등)를 가지고 있는 사람들끼리 서로 유용한 정보를 공유하고 피드백할 수 있는 커뮤니티 아이디어를 기반으로 개발했습니다. <br />
